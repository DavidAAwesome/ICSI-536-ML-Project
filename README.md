# Bone Fracture Image Classifier

## Image Processing Module

Put X-ray images into `/data/input` folder.

Output should generate augmented images in the `/data/output` folder (ignore the blank file in `output` folder, its just so i can add a blank folder).

## Run Tests

Tests are intended to be run with the [pytest](https://docs.pytest.org/en/stable/index.html) framework. Ensure that the package is properly installed in your virtual environment, <u>**and that it is activated**</u>.

Then run:

```
pytest tests
```

from the project root.
